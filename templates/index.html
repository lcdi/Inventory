Flask Boilerplate

Flask Boilerplate is a complete flask install, all it requires is the installation of flask.

Include Packages

Out of the box Flask Boilerplate includes the following front-end packages:

  • Bootstrap
  • JQuery
  • JQuery UI

Bower

I've also included bower, attached to the "static" folder for easy install of additional packages

Open your terminal to the static folder, and use the following command. 

 bower install foo


You can link to the content in src or href attrbutes like you normally would. 

 {% raw %} src="{{url_for('static',filename='bower_components/foo/bar')}}" {% endraw %}
